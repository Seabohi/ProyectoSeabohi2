Clase 8

Lo primero que aprendimos fue la forma de inspeccionar una página web desde el lenguaje HTML, esto se puede hacer mediante la tecla F12 del teclado, para así poder identificar una sección y realizar la modificación si es que lo requiere. Tener en cuenta también que el texto (desde Visual Studio Code) que está en color verde significa que es un comentario y que no se va a visualizar en la página web.

  • Para crear un archivo CSS principal se tiene que enlazar en la parte head del archivo HTML.
  • En HTML se coloca selectores y en CSS se hace una declaración para indicar una funcionalidad.
  • Para modificar a un elemento se utiliza el atributo id=" " y para un conjunto de elementos, el atributo class=" ".
  • Para llamar a una clase en CSS es mediante un punto (.) y a un identificador mediante un numeral (#).
  • Si inspeccionas la página web, se va a localizar un apartado Style que te permite ver la línea del CSS para su modificación.
  • Google Fonts permite descargar fuentes de texto para utilizar en diseño web.
