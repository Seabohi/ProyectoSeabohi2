Clase 1

Inició con la presentación del profesor Óscar Castillo sobre su experiencia profesional y como algunos temas se relacionaban con sus proyectos.También mencionó sus aficiones y las normas de la clase.

Lo siguiente fue la presentación de cada alumno mediante un listado de preguntas, cabe mencionar que la "pregunta rompehielo" era de cualquier tema que se le ocurra al profesor. Fue un momento donde algunos alumnos lo usaron como una especie de terapia ya que comentaron temas muy íntimos al punto de extenderse hasta el final de la clase.

Quedó como pendiente instalar Visual Studio Code y crear una cuenta en GitHub

Clase 2

Se explicó primero sobre la bitácora digital y su importancia en un proyecto. Seguidamente se empezó a explicar el programa Visual Studio Code mientras cada uno abría el programa en su ordenador.

  • Primero se ingresó a GitHub para crear un repositorio.
  • Luego se descargó Git Bash para poder colocar el código de GitHub en Visual Studio Code.
  • Se descarga un plantilla en HTML5UP para poder iniciar la creación de la página.
  • Posteriormente se crea una carpeta en el disco local del ordenador para descomprimir la carpeta de la plantilla.
  • Se reemplaza el archivo index.html para tener las modificaciones de la plantilla descargada.
